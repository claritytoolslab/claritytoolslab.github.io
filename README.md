# claritytoolslab.github.io

**Redirect page** for Clarity Tools Lab organization.

## Purpose

This repository serves as the root GitHub Pages site for the `claritytoolslab` organization. It automatically redirects visitors to the main website:

```
claritytoolslab.github.io → claritytoolslab.github.io/clarity-tools-website/
```

## Implementation

Uses HTML meta refresh for instant redirection:

```html
<meta http-equiv="refresh" content="0; URL='https://claritytoolslab.github.io/clarity-tools-website/'" />
```

## Main Website

The actual Clarity Tools Lab website is located in the [`clarity-tools-website`](https://github.com/claritytoolslab/clarity-tools-website) repository.
